Welcome to Hello Servlet App

This application demonstrates various servlet features including basic HTML output, cookies, sessions, and JSON APIs.

Available Servlets:

  • Hello World Servlet
    Basic servlet that outputs HTML content
  • Cookie Demo
    Demonstrates storing and retrieving data using cookies
  • Cookie Demo (Complete)
    Full implementation of cookie handling with form
  • Session Demo
    Demonstrates storing and retrieving data using HTTP sessions
  • Session Demo (Complete)
    Full implementation of session handling with form
  • Todo API
    Returns a JSON list of todo items (view source to see JSON)

API Documentation:

  • GET /todo-api
    Returns a list of 10 programming-related todo items as JSON
    Example response format:
    [
  {"text":"Implement user authentication system","checked":false},
  {"text":"Fix cross-browser compatibility issues","checked":true}
]
